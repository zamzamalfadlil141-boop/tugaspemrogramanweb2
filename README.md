# Simple CRUD API Express.js

Ini adalah contoh aplikasi CRUD (Create, Read, Update, Delete) sederhana menggunakan Node.js dan Express.js. Aplikasi ini menggunakan *In-Memory Array* sebagai simulasi database.

## Cara Menjalankan

1. Pastikan Node.js sudah terinstal.
2. Clone repository ini: `git clone <url-repo-kamu>`
3. Masuk ke folder project dan instal dependencies:
   ```bash
   npm install
